## Dependencies 
* Zsh
* Tmux
* Vim

## Installation

```sh
$ mkdir -p Repos/dotfiles
$ git clone https://github.com/abdesselame17/dotfiles.git ~/Repos/dotfiles
$ bash ~/Repos/dotfiles/install.sh
```
